Explicação

serve como

A Variável (var) funciona como um contêiner. Ao utilizar var id (identificador) = valor, você está dizendo que id tem esse valor.

Caso eu queira pedir que o usúario escreva 2 números ao entrar no site e que esses números se somem, deve-se tratar a variável como número, não como string. Pois, quando ocorrer var s = num1 + num2, o + poderia ser utilizado tanto para adição, quanto para concatenação. Logo, identidicar as variáveis que irão se somar como número ("Number(n)" ou Numer.parseFloat(n) / Number.parseInt()) ou como string (String(n) ou n.toString())

Formatando String

O uso do string com a variável pode ter 2 versões. A primeira como, por exemplo:

        
var s = 'JavaScript'
'Eu estou aprendendo' + s
E a segunda como: 
        
var s = 'JavaScript'
`Eu estou aprendendo ${s}`
Que éum modo muito mais fácil, pois se usa o placeholder (${})

Aqui vão algumas outras formatações:

  • s.length (contar os caracteres do string)
  • s.toUpperCase() (colocar tudo em letras maiusculas)
  • s.toLowerCase() (colocar tudo em letras minúsculas)

Formatando números

Caso a sua variável seja algum número (Number), pode se formatar das seguintes formas:

  1. Adicionar mais casas decimais 
                    
var n1 = 1543.5
n1.toFixed(2)
1543.50
  2. Adaptar a formatação de dinheiro para a moeda de algum país específico 
                    
n1.toLocalString('pt-br', {style: 'currency', currency: 'BRL'})

Condições

Também aprendi sobre condicionais: if / else. Por exemplo, se eu quero que uma caixa de texto apareça dizendo "Número inválido" quando a resposta de um prompt não for um número, posso fazer:

        
var num = Number(prompt("Digite um número"))
if(isNaN(num)) {
    alert("Número inválido")
} else {}

Ou, caso eu queira fazer com que o código funcione caso uma variável seja maior que a outra:

        
var idd = Number(prompt("Digite sua idade"))
if(idd >= 18) {
    document.write("Bem-vindo ao site!")
} else {
    alert("Acesso negado")
    document.write("Site proibido para menores de 18 anos.")
}

Os operadores irão indicar cálculo. O estudo dos operadores que realizei envolvem 5:

Aritméticos

Contas matemáticas básicas.

  • 2 + 2(Adição)
  • 2 - 1(Subtração)
  • 2 * 2(Multiplicação)
  • 4 / 2(Divisão)
  • 4 % 2(Resto de uma divisão)
  • 4 ** 2(Potenciação)

Caso trabalhar com uma expressão, deve=se sempre utilizar essa ordem de resolução:

  1. ()
  2. **
  3. %; /; *
  4. +; -

Atribuição: simples

É o que atribui valor a uma variável (=). Exemplo:

        
var a = 10
var b = 10 + a
var c = b * a
var d = c - 10
var e = d / 2
var f = e % 3
var g = f ** a

Atribuição: auto-atribuição

Há também como atribuir outro valor à variável. Basta usar os operadores aritméticos para somar, subtrair, multiplicar, dividir, ver o resto da divisão ou potenciar. Ex:

        
var n1 = 20
n1 = n1 + 20

Porém, é possível simplificar esse cálculo como apenas:

        
var n1 = 20
n1 += 20

E para somar ou diminuir somente 1 no valor:

        
var n1 = 20
n1++ //ou ++n1 //Me explique a diferença, Copilot.

Relacionais

Todos aqueles que buscam estabelecer uma relação binária e verificar o resultado booleano (true, false). Seus resultados serão: true ou false.

Operadores

  • >(Maior que)
  • <(Menor que)
  • >=(Maior ou igual a)
  • <=(Menor ou igual a)
  • ==(Igual a)
  • !=(Diferente de)

Relacionais de identidade

O operador == analisa se o um valor é igual a outro. Porém, caso a operação seja 5 == '5', o valor também dará true, pois o esse operador não analisa os tipos de dados. Para tal, temos esse outro operador, que pode até ser usado com !=:

        
5 === '5'
false
//para dizer que um número é diferente que outro:
var n1 = 10
var n2 = '10'
n1 != n2
//Seria false

//Porém e se eu escrever assim?
var n1 = 10
var n2 = '10'
n1 !== 10
//Seria true

Lógicos

Temos 3 tipos:

  • !(Negação)
  • &&(Operador "E" binário. Serve para condição binária, que só se torna verdade quando os 2 valores são true)
  • ||(Operador "OU" binário. Serve para condição binária que se torna true caso uma das condições sejam true)

A ordem de cálculo para Operadores (Aritméticos > Relacionais > Lógicos>); para os lógicos (! > && > ||).
